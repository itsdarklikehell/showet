# Platforms: Commodore

## Supported emulators:

```python
emulators = ['retroarch', 'zesarux']
```

## Supported Libretro Cores:

```python
cores = ['crocods_libretro', 'cap32_libretro']
```

## Supported file extensions:

```python
crocods_libretro:
extensions = ['zip', 'dsk', 'sna', 'kcr']

cap32_libretro:
extensions = ['zip', 'dsk', 'sna', 'zip', 'tap', 'cdt', 'voc', 'cpr', 'm3u']
```
