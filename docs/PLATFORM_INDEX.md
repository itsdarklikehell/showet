# Platform Documentation Index

Welcome to the Showet Platform Documentation. Each platform has been carefully integrated with period-authentic settings.

## Supported Platforms

| Platform | Core | Authentic Era | Recommended Shader |
|----------|------|---------------|-------------------|
| Commodore 64 | vice_x64sc | 1980s Home Computing | CRT-Easymode |
| Commodore Amiga | vice_x64 | 1990s Demo Scene | CRT-Royale |
| MS-DOS PC | dos | VGA Demoscene | CRT-PI |
| Nintendo Famicom | quicknes | 8-bit Console | CRT-Easymode |
| Sega Megadrive | genesis_plus_gx | 16-bit Console | CRT-Easymode |
| Atari 2600 | stella | 8-bit Arcade | CRT-PI |
| Sony PlayStation | mednafen_psx | 32-bit CD Era | CRT-Royale |

## Nostalgia Integration

Each platform automatically loads its authentic sound theme when launched in Museum Mode. Press `S` in the viewer to toggle sound themes.

## Configuration Files

All platform configs are in `nostalgist_configs/` with period-appropriate CRT shaders pre-selected.
